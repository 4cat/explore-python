关于本书

思维导图？？ 目录？

本文用的 Python 版本是 2.7.11，开发环境 mscOS

本书的定位：不是一本 Python 的教程，而是对 Python 常用概念的介绍，每章基本都是独立的，读者可以挑选感兴趣的章节进行学习。

本书主要分为 15 个章节：

  • 第 1 章：介绍一些基础知识，包括 Python 中的输入和输出，字符编码。
  • 第 2 章：介绍 Python 中的基本数据类型，比如字符串、列表和字典等。
  • 第 3 章：介绍 Python 中的函数。
  • 第 4 章：介绍 Python 中的函数式编程，包括闭包和装饰器等。
  • 第 5 章：介绍 Python 中类的使用，以及一些特殊方法。
  • 第 6 章：介绍 Python 中的高级特性，比如生成器，上下文管理器。
  • 第 7 章：文件和目录操作。
  • 第 8 章：进行、线程和协程。
  • 第 9 章：错误和异常处理。
  • 第 10 章：正则表达式。
  • 第 11 章：HTTP 服务。
  • 第 12 章：测试。
  • 第 13 章：一些标准模块的使用。
  • 第 14 章：一些第三方模块的使用。
  • 第 15 章：结束语。

本书仍在持续更新中，如果有问题，请及时向我反馈。
